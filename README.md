# 📷 Smart Camera Control App (Android + ESP32 | BLE + WiFi)

## 🚀 Overview
Android application to control and monitor an ESP32-based camera system using BLE for provisioning and WiFi hotspot for real-time data streaming.

---

## ⚙️ Features
- BLE device discovery and pairing
- GATT-based provisioning for device setup
- Seamless switch from BLE → WiFi hotspot
- Real-time camera control and data streaming
- Stable and low-latency communication

---

## 🛠️ Tech Stack
- Kotlin, Android SDK
- BLE (GATT), WiFi (Hotspot)
- ESP32
- MVVM Architecture

---

## 📡 Workflow
1. Scan & connect via BLE  
2. Configure device using GATT  
3. Switch to WiFi hotspot  
4. Stream real-time data  
5. Control device via app
